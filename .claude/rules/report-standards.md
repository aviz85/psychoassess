# Psychological Report Standards

These standards apply to all report writing within this workspace.

## Formatting Standards

### Required Sections
1. Identifying Information
2. Reason for Referral
3. Background Information
4. Behavioral Observations
5. Assessment Results
6. Integrated Summary
7. Diagnostic Impressions
8. Recommendations

### Score Presentation
- Always include **confidence intervals** (95% preferred)
- Include **percentile ranks** with all standard scores
- Use **consistent classification system** (Wechsler)
- Present in **table format** for clarity

## Writing Standards

### Tone
- **Professional** but accessible
- **Objective** and balanced
- **Respectful** of client and family
- **Strength-based** framing when possible

### Language
- Explain **technical terms** when first used
- Use **parent-friendly** language
- Maintain **consistent terminology**
- Avoid **jargon** without explanation

### Structure
- Use **clear headings** and subheadings
- Present information **logically**
- Connect findings to **referral questions**
- Link recommendations to **findings**

## Content Standards

### Behavioral Observations
Must include:
- Physical presentation
- Rapport/social behavior
- Attention and concentration
- Approach to tasks
- Validity statement

### Test Results
Must include:
- Full test name and abbreviation
- Brief description of what test measures
- All relevant scores with CIs and percentiles
- Interpretation connected to referral

### Diagnostic Impressions
Must include:
- Clear statement of diagnostic conclusions
- Supporting rationale for each impression
- Conditions considered and ruled out
- Appropriate diagnostic codes (if applicable)

### Recommendations
Must be:
- **Specific** and actionable
- **Connected** to findings
- **Prioritized** by importance
- **Implementable** in real settings
- **Comprehensive** but not overwhelming

## Quality Checklist

Before report completion:
- [ ] All referral questions addressed
- [ ] All scores accurate and verified
- [ ] Confidence intervals included
- [ ] Classifications consistent
- [ ] Behavioral observations support interpretation
- [ ] Technical terms explained
- [ ] Recommendations linked to findings
- [ ] Client identifiers consistent and accurate
- [ ] Dates accurate
- [ ] Examiner credentials included
- [ ] Report proofread for errors
- [ ] Appropriate hedging used throughout
