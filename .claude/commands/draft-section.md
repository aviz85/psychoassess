# Report Section Drafting Command

Draft the "$1" section of the psychological report.

## Instructions

Use the report-writing skill to:

1. **Follow section template** - Use appropriate structure for this section
2. **Use professional language** - Reference phrase bank for wording options
3. **Maintain consistency** - Match report style and terminology
4. **Include hedging** - Use appropriate clinical language
5. **Be parent-friendly** - Explain technical terms

## Available Sections

### background
- Developmental history
- Medical history
- Educational history
- Social/emotional history
- Family information

### observations
- Physical presentation
- Rapport development
- Attention/concentration
- Approach to tasks
- Validity statement

### results
- Test-by-test results
- Score tables with confidence intervals
- Interpretation of each measure
- Cross-test patterns

### summary
- Integrated summary of findings
- How findings address referral questions
- Diagnostic impressions with rationale
- Convergent evidence synthesis

### recommendations
- Educational accommodations
- Academic interventions
- Home supports
- Follow-up needs
- Prioritized action items

## Interactive Mode

Before drafting, I may ask you for:
- Specific data to include
- Scores or observations
- Referral questions to address
- Preferred emphasis or framing

After drafting, I can:
- Revise specific sections
- Add more detail
- Adjust language/tone
- Integrate additional data
